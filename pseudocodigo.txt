(7). ->  Hacer el pseudocódigo de un programa que permita escribir los 100 primeros pares.
1. Inicializar contador a 0

     a. Mostrar número
     b. número = número + 2

(8). -> Diseña el pseudocódigo necesario para sumar los N primeros impares. Realizar después
uno que haga lo mismo con los pares y otro con los múltiplos de 3. Crea un menú para
ello.

1. Mostrar menú:
   a. Opción 1: Sumar N primeros impares
   b. Opción 2: Sumar N primeros pares
   c. Opción 3: Sumar N primeros múltiplos de 3
   d. Opción 4: Salir
2. Leer opción desde teclado
3. Si opción es igual a 1:
     a. Leer N desde teclado
     b. Inicializar suma a 0
     c. Inicializar número a 1
     d. Para i desde 1 hasta N:
          i. suma = suma + número
          ii. número = número + 2
     e. Mostrar suma
4. Si opción es igual a 2:
     a. Leer N desde teclado
     b. Inicializar suma a 0
     c. Inicializar número a 2
     d. Para i desde 1 hasta N:
          i. suma = suma + número
          ii. número = número + 2
     e. Mostrar suma
5. Si opción es igual a 3:
     a. Leer N desde teclado
     b. Inicializar suma a 0
     c. Inicializar número a 3
     d. Para i desde 1 hasta N:
          i. suma = suma + número
          ii. número = número + 3
     e. Mostrar suma
6. Si opción es igual a 4, salir del programa
7. Repetir desde el paso 1
